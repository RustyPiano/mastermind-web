# 密码机 (Mastermind)

🔐 一个受 **GiiKER 计客超级密码机** 启发的极简风格网页版猜色游戏，支持每日挑战、单人经典、重复色变体与双人对战四种模式。

## ✨ 特性

- **四种模式**：每日挑战（全球同题，计入连胜）、单人经典（练习与刷新最佳步数）、重复色变体（允许重复颜色，难度更高）、双人对战（同设备 Pass-and-Play）。
- **统计系统**：追踪总场次、胜率、每日连胜 / 最佳连胜、经典最佳步数与平均步数。
- **自动存档**：游戏进度自动写入 `localStorage`，刷新或关闭页面后可无缝恢复。
- **分享结果**：优先调用系统原生分享（移动端），降级为复制到剪贴板，Emoji 格式不包含颜色信息防剧透，并附带官网链接。
- **极简深色美学**：受 iOS/macOS 深色模式启发，克制配色专注游戏本身。
- **响应式布局**：完整适配手机与桌面，桌面端为棋盘在左、操作区在右的双栏布局；手机端维持上下分屏，猜测阶段棋盘历史可滚动。
- **零外部依赖**：纯 Vanilla JS（ES Modules）+ CSS3 + 语义化 HTML5，无任何生产依赖。

## 🎮 游戏规则

**目标**：在 10 次内猜出 4 个颜色组成的密码（顺序也要正确）。

可用颜色共 7 种：红、橙、黄、绿、蓝、紫、白。

每次提交猜测后，系统给出反馈点：

| 反馈 | 含义 |
|------|------|
| 🟢 绿色点 | 颜色正确，且位置也正确 |
| 🟠 橙色点 | 颜色正确，但位置错误 |
| ⚪ 空心点 | 该颜色不在密码中 |

> 反馈点的数量对应线索数，但**不指示具体是哪个位置**。

### 模式说明

| 模式 | 密码来源 | 允许重复色 | 机会次数 | 特点 |
|------|----------|-----------|---------|------|
| 每日挑战 | 当天固定密码（全球同题） | 否 | 10 | 通关计入每日连胜 |
| 单人经典 | 电脑随机生成 | 否 | 10 | 追踪最佳步数与平均步数 |
| 重复色模式 | 电脑随机生成 | 是 | 10 | 推理难度更高 |
| 双人对战 | 玩家一手动设置 | 否 | 10 | 同设备轮流，玩家一出题玩家二破解 |

## 🚀 运行方式

由于项目使用 ES Modules，需通过本地 HTTP 服务器启动（不能直接双击 HTML）。

```bash
# 方式一：Python 3（无需安装额外工具）
python3 -m http.server 3000

# 方式二：Node.js npx
npx serve .
```

启动后访问 `http://localhost:3000`。

也可以直接用支持 ES Modules 的开发工具（如 VS Code Live Server）打开。

## 🧪 测试

```bash
npm install
npm test
```

使用 [Vitest](https://vitest.dev/) 覆盖核心逻辑：游戏引擎、每日挑战生成、统计系统、状态持久化、分享文本、模式配置。
