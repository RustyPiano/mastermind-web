# 密码机 (Mastermind)

🔐 一个受 **GiiKER 计客超级密码机** 启发的极简风格网页版猜色游戏，支持单人与双人模式。挑战你的逻辑推理与破解能力！

## ✨ 特性 (Features)

- **极简深色美学 (Minimal Dark)**：受 iOS/macOS 深色模式启发，克制的高级灰纯色背景与高亮强调色，专注游戏本身。
- **顺滑的焦点交互**：支持“光标焦点”模式，玩家可自由点击任意孔位进行单独修改、填色或取消已填颜色，操作符合直觉。
- **双模式玩法**：支持「单人模式」（电脑随机生成密码）与「双人模式」（玩家一设置，玩家二破解）。
- **纯原生实现**：0 外部依赖！使用纯 Vanilla JS (ES Modules)、CSS3 (变量与现代布局) 和语义化 HTML5 构建。
- **响应式设计 (Mobile First)**：完美适配各种尺寸的手机屏幕与桌面浏览器，支持触摸设备的防误触优化。
- **SEO 与无障碍友好**：完整的 Meta 标签、Open Graph 分享标签以及基础的 ARIA 支持。
- **优化配色与布局**：7 色调色板采用 4+3 平衡网格，包含高对比白色球，移动端棋盘视觉更居中。

## 🎮 游戏规则 (How to Play)

游戏开始后可选择模式：

- **单人模式**：电脑随机生成 4 位互不重复的密码，你负责破解。
- **双人模式**：玩家一设置密码，玩家二在同一设备上破解（Pass-and-Play）。

可用颜色共 7 种：红、橙、黄、绿、蓝、紫、白。

### 阶段一：设置密码 (玩家一)
1. 玩家一从 7 种颜色中挑选 **4 种互不相同的颜色**，排列成一组密码。
2. 确认密码后，屏幕将被锁定，此时将设备交给玩家二。

### 阶段二：破解密码 (玩家二)
1. 玩家二接手设备，尝试猜出玩家一设置的 4 个颜色及其排列顺序。
2. 每次提交猜测后，系统会给出线索反馈（反馈点不对应具体位置）：
   - 🟢 **绿色反馈点**：表示有 1 个颜色 **选对了，且位置也完全正确**。
   - 🟠 **橙色反馈点**：表示有 1 个颜色 **选对了，但位置放错了**。
   - ⚪ **空心边框**：表示该颜色根本不在密码中。
3. 玩家二共有 **10 次** 猜测机会。在 10 次内获得 4 个绿色反馈即为破解成功，否则挑战失败！

## 🚀 如何运行 (How to Run)

由于本项目使用了现代浏览器的 ES Modules (`<script type="module">`) 特性，直接双击打开 HTML 文件可能会遇到 CORS 跨域限制。

请使用任意本地 HTTP 服务器启动：

**方法 1：使用 Node.js (npx)**
```bash
npx serve .
```

**方法 2：使用 Python**
```bash
# Python 3
python3 -m http.server 3000
```

启动后，在浏览器中访问 `http://localhost:3000` 即可开始游戏。
